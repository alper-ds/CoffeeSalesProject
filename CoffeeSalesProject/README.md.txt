# Coffee Sales Analysis

## Project Overview
This project analyzes coffee shop sales data, including sales by coffee type, hourly trends, weekday & monthly sales, and payment method distribution.

## Dataset
This dataset was obtained from [Kaggle](https://www.kaggle.com/) and contains coffee shop transaction records.

## Features
- hour_of_day: Hour of purchase (0â€“23)
- cash_type: Payment method (cash / card)
- money: Transaction amount
- coffee_name: Coffee type
- Time_of_Day: Morning / Afternoon / Night
- Weekday / Weekdaysort: Day of the week
- Month_name / Monthsort: Month of purchase
- Date / Time: Exact transaction date and time

## Visualization
- Total revenue
- Sales by coffee type
- Sales by hour
- Sales by weekday
- Sales by month
- Payment type distribution

## How to Run
1. Install dependencies: `pandas`, `matplotlib`, `seaborn`, `numpy`
2. Run `coffee_sales_analysis.py`
